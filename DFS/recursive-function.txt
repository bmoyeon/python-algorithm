재귀함수와 스택

재귀함수는 자기자신을 반환하는 함수이다.
또한 재귀함수는 스택 자료구조를 사용한다.

아래 예를 살펴보면 확연히 알 수 있을 것이다.
def func(n):
    if n > 0:
        print(n, end=' ')
        func(n-1)

func(3)

위 함수의 실행결과는
3 2 1

그러나 코드 순서를 뒤집어 print 문을 맨 마지막에 둔다면 결과는 어떻게 바뀔까?

def func(n):
    if n > 0:
        func(n-1)
        print(n, end=' ')

func(3)

이제 위 함수의 실행결과는 아래와 같이 출력된다.
1 2 3

왜 결과가 다를까?
위에서도 말했듯이 재귀함수는 스택 자료구조를 사용하기 때문이다.
함수를 호출하면서 스택에 데이터가 들어가게 된다.
func(3)을 호출하면 함수가 바로 func(2)를 호출한다.
그럼 func(2)는 func(1)을 호출할 것이다.
n이 0이 되어 끝나면 21번째 줄로 돌아와 print문이 실행된다.
스택에 있는 데이터가 차례대로 지워지게 된다.
